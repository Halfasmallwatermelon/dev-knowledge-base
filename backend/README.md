# 后端

API 设计、业务架构、认证授权、中间件、消息队列、微服务等笔记存放于此。

## 按语言分类

| 目录 | 说明 | 常见框架 |
|------|------|----------|
| [node/](node/) | JavaScript / TypeScript | Express、Fastify、NestJS |
| [python/](python/) | Python | FastAPI、Django、Flask |
| [go/](go/) | Go | Gin、Echo、标准库 net/http |
| [java/](java/) | Java | Spring Boot、Quarkus |
| [csharp/](csharp/) | C# / .NET | ASP.NET Core、EF Core、JWT、缓存、后台任务 |
| [rust/](rust/) | Rust | Axum、Actix Web |

语言无关的通用主题（REST 设计、认证模式、微服务架构等）可直接放在本目录下。
